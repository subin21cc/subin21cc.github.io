# 페르소나

27세 직장인 김민수 씨는 최근 검진에서 당뇨 위험 판정을 받았다. 아침은 편의점, 점심은 구내식당, 저녁은 배달음식이 일상이다. 삼성헬스를 써봤지만 수동 입력이 귀찮아 이틀 만에 포기했고, 어떤 음식이 혈당을 올리는지 알고 싶지만 기존 앱은 칼로리만 알려준다.

# 사용자 액션 시나리오

### 1. [대시보드]

- **건강 상태 모니터링**: 민수 씨는 출근길에 앱을 켜고 대시보드 상단의 **칼로리, 나트륨, 혈당** 프로그레스 바를 확인하여 오늘 남은 영양 섭취 허용량을 체크합니다.
- **일정 확인**: '오늘의 일정' 섹션에서 오후에 예약된 병원 방문 시간을 다시 한번 확인합니다.

### 2. [식단]

- **카메라 호출**: 저녁 식사 전, 화면 중앙의 **[카메라] 버튼**을 눌러 촬영 모드로 진입합니다.
- **식단 촬영**: 배달된 비빔밥 사진을 촬영합니다. 이때 시스템은 음식 이미지만을 인식하여 자동으로 분석을 시작합니다.
- **영양 정보 확인 및 보정**: 분석 결과 창에 뜬 **나트륨 함량**과 영양 성분을 확인합니다. 국의 종류가 잘못 인식되었다면 **[수동 보정]** 버튼을 눌러 수동 조정한 뒤 [저장하기]를 누릅니다.

### 3. [운동]

- **주변 센터 검색**: **[플레이스]** 탭을 눌러 집 근처 헬스장 위치와 리스트를 확인합니다.
- **트레이너 상담 요청**: 마음에 드는 센터를 고른 뒤 **[트레이너 1:1 채팅]** 버튼을 클릭합니다.
- **데이터 공유 및 대화**: 민수 씨의 식단과 운동 로그가 요약된 '건강 데이터 요약본'이 트레이너에게 전송된 것을 확인하고, 별도의 설명 없이 바로 "오늘은 어떤 운동을 하면 좋을까요?"라고 질문하며 상담을 시작합니다.

### 4. [MY]

- **건강 지표 체크**: 하루 일과를 마치고 **[마이]** 탭에 접속하여 체중과 혈당 수치가 기록된 **변화 그래프**를 살펴보며 성취감을 느낍니다.
- **포인트 적립**: 오늘 모든 식단과 운동을 기록하여 지급된 **포인트**를 확인합니다.
