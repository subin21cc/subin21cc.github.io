# 6장. 깃허브에서 다른 사람과 소통하기

## 6-1. 깃허브 프로필 관리하기

### 프로필 작성하기

‘https://github.com/아이디’로 접속하면 해당 아이디 사용자의 프로필 사진을 비롯해 저장소 목록과 커밋 현황 등 다양한 정보를 보여주는 화면 나타남

### 컨트리뷰션 살펴보기

- 컨트리뷰션 그래프(contribution graph): 깃허브 프로필 화면에서 초록색 사각형이 찍힌 부분
  - 깃허브 사용자들끼리는 컨트리뷰션 그래프에 초록색 사각형을 채워 가는 것을 ‘잔디를 심는다’라고 얘기하곤 한다.

---

## 6-2. README 파일 작성하면서 마크다운 익히기

- README 파일: 해당 저장소를 소개하는 공간
- 마크다운 문법을 사용하므로 확장자는 .md를 사용한다.

### README에 이미지 추가하기

```markdown
![대체 텍스트](이미지 파일 경로)
```

---

## 6-3. 오픈 소스 프로젝트에 기여하기

- 컨트리뷰션에는 소스 코드의 버그를 수정하는 것 뿐만 아니라 그 오픈 소스에 도움이 될 만한 거의 모든 활동이 포함된다.
