# 8장. 깃허브에 이력서 사이트와 블로그 만들기

## 8-1. HTML, CSS 파일로 이력서 사이트 만들기

- 사용할 파일 준비하기
  - 첫 화면은 반드시 index.html로 저장해야 한다
- 다른 사람의 저장소 복제하기 - fork
- 지역 저장소로 복제하기 - clone
- 사용할 이미지 파일 교체하기
- index.html 파일 수정하기
- 변경 사항 커밋하기
- 원격 저장소를 페이지로 전환하기

- 지킬 테마 살펴보기
- 블로그 만들고 기본 환경 설정하기
- 블로그에 포스트 작성하기
- 블로그 포스트에 이미지 넣기
